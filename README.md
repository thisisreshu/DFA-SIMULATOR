# DFA-SIMULATOR
This project is a simulation of a Deterministic Finite Automaton (DFA) from the subject Theory of Computation. The application allows users to enter input strings and observe how the automaton processes the string through different states using a state transition function.

The simulator demonstrates the working of a DFA by showing state transitions step-by-step based on the given alphabet and transition rules. It finally determines whether the input string is accepted or rejected by the automaton according to the defined final states.

This project helps students understand the concept of DFA, state transitions, alphabets, and languages in an interactive and visual way.
